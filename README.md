# FB Reels Cleaner

A free Chrome extension to delete all your Facebook Reels at once — 
or one by one automatically. No subscriptions, no login required.

## 🔗 Full Guide & Download
👉 [stacknix.dev/delete-all-facebook-reels-at-once](https://stacknix.dev/delete-all-facebook-reels-at-once/)

## How to Install

1. Download this repository as ZIP (green Code button → Download ZIP)
2. Extract the ZIP on your computer
3. Open Chrome → go to `chrome://extensions`
4. Enable **Developer Mode** (top right toggle)
5. Click **Load unpacked** → select the extracted folder
6. Open Facebook and click the extension icon

## Features

- ⚡ Delete ALL reels at once via Activity Log
- ↩ Delete one reel and jump to next
- ▶ Auto-delete while viewing reels
- 🔍 Debug scan for troubleshooting
- ⬛ Stop anytime
